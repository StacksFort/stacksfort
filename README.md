# StacksFort

A decentralized multisig vault platform built on the Stacks blockchain.

## Structure

```
stacksfort/
├── contract/          # Clarity smart contracts
└── frontend/          # Next.js web application
```

## Quick Start

### Smart Contracts
```bash
cd contract
npm install
clarinet test
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
